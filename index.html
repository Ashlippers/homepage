APRS Weather Box

Brief Introduction

A simple Arduino project that collects basic weather data and sends WX (weather) beacons to APRS (Automatic Packet Reporting System) network as a simple APRS weather station. A nice handy project for amateur radio operators (HAMs).

Coded by Weng Kai (BA5AG) and me. Source code here.

简介

APRS Weather Box是一个简单的Arduino小程序，收集气象信息并向APRS网络发送气象信标，作为简易的APRS气象站。

代码由翁恺老师（BA5AG）和我编写。源代码在此处。
